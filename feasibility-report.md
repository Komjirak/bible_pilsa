# 적정성 검토 보고서 — 말씀필사 (Bible-Pilsa)

> 작성일: 2026년 3월 6일
> 서비스 아이디어: 40~60대 기독교인 대상 성경 말씀 일일 필사 챌린지 앱
> 핵심 BM: 주 7일 필사 완주 → 토스 포인트 10원 지급 + 인앱 광고(IAA) 수익

## 2. 수익화 가능 여부

| 항목 | 판정 | 세부 내용 |
|------|------|----------|
| 수익화 기능 필요 여부 | ✅ | 토스 포인트 지급 + IAA 광고 수익 병행 |
| 사업자 등록 여부 | ⚠️ | **토스 포인트 지급 및 IAA 광고 수익화는 사업자 필수** |
| 토스 포인트 지급 | ⚠️ | 사업자만 사용 가능 — Point API 필요 |
| IAA 광고 수익 | ⚠️ | 사업자만 사용 가능 — 인앱광고 계약 필요 |
| 대안 수익화 방식 | ✅ | 비사업자라면 자체 스탬프/배지 시스템으로 대체 가능 |

**BM 상세 분석**:

### 수익 구조

| 수익원 | 방식 | 예상 단가 |
|--------|------|----------|
| IAA (인앱광고) | 전면형 광고 (필사 완료 후 노출) | CPM 기준 약 3,000~8,000원/1,000회 |
| IAA (인앱광고) | 보상형 광고 (포인트 적립 전 선택 시청) | CPM 기준 약 15,000~30,000원/1,000회 |
| 포인트 지급 비용 | 주 7일 완주 시 10원 (사용자당) | 비용 항목 |

### BM 타당성 계산 (월 기준, 보수적 추정)

```
MAU 10,000명 가정
  - 주 7일 완주율 30% = 3,000명 × 10원 × 4주 = 120,000원 포인트 비용
  - 일일 접속 필사 화면 광고 노출: 10,000 DAU × 1회 광고 × 30일
    → 300,000 임프레션 × CPM 5,000원/1,000 = 1,500,000원 광고 수익
  - 보상형 광고 (완주자 전원 시청): 3,000명/주 × 4주 = 12,000회
    → 12,000 × CPM 20,000원/1,000 = 240,000원 추가

월 광고 수익(추정): 약 1,740,000원
월 포인트 비용(추정): 약 120,000원
순이익(추정): 약 1,620,000원
```

> MAU 10만 명 달성 시 월 1,600만 원 수준의 광고 수익 기대 가능
> 40~60대 기독교인 광고 CPM은 구매력 있는 연령대로 상단 보정 가능성 높음

**결론**: ⚠️ **사업자 등록이 전제 조건** — 사업자 확인 후 진행 필수

**결론**: ✅ 기술적으로 타당 — 개역한글판 활용 + BE 구성 필요

| 항목 | 판정 | 세부 내용 |

| 핵심 타겟 규모 | ✅ | 한국 기독교인 약 900만 명, 40~60대 약 40% → **약 360만 명** 잠재 사용자 |
| 재방문 유도 메커니즘 | ✅ | **일일 루프 최강** — 매일 말씀 필사 → 주간 완주 보상 체계 |
| 토스 컨텍스트 접점 | ✅ | 40~60대 토스 사용자 + 포인트 적립 동기 → 자연스러운 접점 |
| 경쟁 서비스 유사 사례 | ✅ | 성경 필사 앱(갓피플, Bible365 등) 존재 — 포인트 보상 차별화 가능 |
| 시장 독특성 | ✅ | 성경 필사 + 토스 포인트 보상 조합은 현재 국내 미존재 |

### 사용자 행동 패턴 분석 (40~60대 기독교인)

```
[일일 루프]
아침 기도/큐티 습관 → 앱 접속 → 오늘의 말씀 1절 확인 → 필사
→ 완료 체크 → 주간 진행률 확인 → (광고 노출) → 앱 종료

[주간 보상 루프]
7일 완주 → 토스 포인트 10원 적립 → 성취감 + 금전 보상 →
다음 주 계속 참여 동기 부여

[소셜 루프 (선택)]
필사 완료 인증 공유 → 교회 지인에게 전파 → 바이럴
```

**리텐션 강점**: 기독교인의 큐티(경건의 시간) 습관과 필사 결합 → 종교적 의무감과 게이미피케이션의 시너지

**결론**: ✅ 리텐션 구조 매우 강함, 시장 규모 충분

## 종합 권고

> **개발 진행** — 단, 두 가지 전제 조건 확인 후 진행

**권고 이유**:
- 리텐션 구조(일일 필사 루프)가 매우 탄탄하고, 성경 데이터는 퍼블릭 도메인(개역한글판)으로 저작권 이슈 해결 가능
- 토스 포인트 + 광고 BM이 수익성 측면에서 현실적이며 계산 가능한 범위
- 국내 경쟁 앱 대비 "포인트 보상 필사" 차별화 포인트 명확

**진행 전제 조건**:
1. **사업자 등록 여부 확인** — 포인트 지급 및 IAA 광고 수익화는 사업자 필수. 비사업자라면 포인트 지급 없이 배지·스탬프 보상으로 변경해야 함
2. **개인정보처리방침 사전 준비** — 필사 이력 저장에 따른 법적 의무

**대안 제안** (비사업자인 경우):
- 토스 포인트 → **자체 스탬프 수집 + 레벨업 시스템**으로 대체 (즉시 개발 가능)
- 이 경우 BE 의존도 대폭 감소 → FE 전용 구현 가능 → 개발 복잡도 낮음으로 하향
- 광고 수익화는 사업자 등록 후 단계적 도입 가능

**성경 번역본 권고**:
- **MVP**: 개역한글판 번들 내장 (무료, 즉시 사용 가능) + "대한성서공회" 출처 표기
- **성장 단계**: 대한성서공회와 개역개정판 라이선스 협의 (유료 가능성 있음)
