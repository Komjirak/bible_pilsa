# Changelog - 말씀필사 (Komjirak Bible)

## [1.0.2] - 2026-03-24

### 🚀 추가 및 개선 (Features & Enhancements)
- **"오늘의 말씀" 매일 랜덤 선택 로직 적용**
  - DJB2 해시 기반 의사 난수 알고리즘으로 매일 다른 구절이 랜덤하게 표시
  - 같은 날 동일 구절 보장 (결정론적), 날마다 랜덤 변경
  - 백엔드: 31,102절 전체 성경 데이터 풀 기반 랜덤 선택
  - 프론트엔드: 샘플 데이터 풀 해시 기반 랜덤 선택으로 동기화
- **당일 무제한 필사 진도 진행 로직 개편**
  - 1일 1회 달란트 보상 지급 이후에도 사용자가 원할 경우 이어서 필사 가능하도록 설계
  - `randomOffset` 글로벌 상태 관리를 통해 "오늘의 말씀" 연속 필사 시마다 새로운 랜덤 구절 노출
  - 홈 화면 완료 버튼 텍스트("이어서 필사하기") UX 개선을 통한 연속 행동 유도
- **토스 미니앱 (`.ait`) 정식 빌드 파이프라인 정립**
  - `apps-in-toss` 공식 CLI를 활용한 빌드 환경 구축으로 앱 구동 안정성 확보

## [1.0.1] - 2026-03-21

### 🚀 추가 및 개선 (Features & Enhancements)
- **토스 프로모션 SDK 연동 (`grantPromotionReward`)**
  - 가이드 기반의 정식 `@apps-in-toss/web-framework` 모듈 주입 및 중복 방어
  - 프로모션 예산 소진(4109), 이미 지급(4110), 기간 외(4111) 등 정밀 에러 제어 및 분기 화면 가이드 출력
- **서비스 소개 페이지 (IntroPage) UI 리노베이션**
  - 자이언트 앱 아이콘을 전면 중앙 집중형으로 배치
  - 세로 타임라인 레이아웃 채용을 통한 연동 흐름 시각화 향상
- **주간/연속 필사 보상 로직 분리 및 고도화**
  - `Sequential` 모드 연속 필사 시 동일일자 무제한 진도 진행 가능 개방
  - 연속 필사 재진입 시 "추가 필사 격려" 배너 및 달란트 unique 라우팅 추가

### 🛠 UI/UX 폴리싱 (Polishes)
- **앱내 화폐 명칭 전수 개명:** `"포인트"` ➡️ 브랜딩 화폐인 `"달란트"`로 전면 개명 완료
- **홈카드 책갈피 보완:** 오우버래핑(Overlap) 방지용 Safe-Area 패딩 삽입
- **자동 컨펌 알림 전환:** 설정 화면 푸시 피커 비동기 딜레이 삭제 (셀렉트 즉시 얼럿 확인)
- **개발자 전용 격리 도구 (`/dev`):** 연속성 강제 치환 및 초기화 마이크로 헬퍼 세팅

## [1.0.0] - 2026-03-21 (초기 릴리즈)

### ⚙️ 코어 아키텍처 (Architecture)
- **Zustand `useProgressStore` 영속성 결합:** `localStorage` 기반 백업 처리
- **앱인토스 최적화 빌드 빌딩:** Static Import 전개 구성
- **서버리스 비중 증가:** 1,000절 성경 인덱스 로컬화 탑재 연동

### 📺 기능 구현 (Functions)
- **데일리 필사 가이드 검증:** 오타 검출(Red-overlay) 백트래킹 및 폰트 축소/확장 렌더링
- **토스 광고 연동 완료:** 홈화면 및 클로징 화면 `BannerAd`, 완성 진입 시 `FullScreenAd` 장착
