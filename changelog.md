# Changelog - 말씀필사 (Komjirak Bible)

## [1.0.4] - 2026-03-30

### 🛠 버그 픽스 (Bug Fixes)
- **안드로이드 OS 하드웨어 폰트 스케일링 완전 방어 (근본 수정)**
  - **원인:** 기존 `text-size-adjust: 100%` 및 `clamp()` CSS 방어는 브라우저 자체의 텍스트 자동 크기 조절만 비활성화할 뿐, 안드로이드 OS 접근성 설정(`Settings > Display > Font Size`)이 WebView `textZoom`을 통해 CSS `px` 단위 자체를 스케일링하는 것을 차단하지 못함
  - **해결:** JavaScript 런타임에서 실제 렌더링된 16px 요소의 물리적 높이를 측정하여 OS 스케일 팩터를 역산하고, `#root`에 CSS `zoom` 역보정을 적용하는 `fontScaleGuard` 유틸리티 도입
  - `index.html`에 인라인 스크립트로 번들 로딩 전 즉시 역보정을 적용하여 초기 로딩 시에도 레이아웃 깨짐 방지
  - `text-size-adjust: none !important` 글로벌 적용으로 이중 방어
  - `fontScaleGuard` 코드 리팩토링: `zoom` 보정, `html.width/height` 확장, `body.justifyContent` 고정, `#root.maxWidth` 해제, `--screen-height` 변수 추가

## [1.0.3] - 2026-03-27

### 🚀 추가 및 개선 (Features & Enhancements)
- **홈 화면 모드 탭(오늘의 말씀/순서대로) 기억 기능**
  - 앱 재시작 시에도 사용자가 마지막으로 선택한 탭을 `localStorage` 기반으로 유지하도록 보완
- **필사 완료 화면 시각적 사용성 최적화**
  - 하측 대형 배너광고(`image-highlight` Variant - 250px) 탑재에 따른 레이아웃 압축 진행
  - 상하 여백 및 완료 아이콘 사이즈 축소를 통해, 소형 모바일 기기에서도 "확인" 버튼이 한 화면에 온전히 들어오도록 스크롤 동선 개선

### 🛠 버그 픽스 (Bug Fixes)
- **오타 수정 시 커서(깜빡이) 위치 어긋남 현상 완벽 해결**
  - 투명 `textarea` 입력칸에 `fontFamily: 'inherit'` 속성을 강제로 주입하여, 오버레이 폰트(Pretendard)와 글자 장평을 100% 일치시켜 커서 오동작 원천 차단
- **안드로이드 기기 하드웨어 폰트 스케일링 방어**
  - `text-size-adjust: 100%` 속성 및 `clamp(14px, 16px, 18px)`를 적용하여, 기기 설정때문에 UI 레이아웃이 무너지는 현상 방지
- **자정 직후 스탬프 체크(✔️) 누락 현상 고침**
  - 로컬 스토어의 날짜 계산이 UTC(영국 시간) 기준으로 실행되던 버그를 확인하고, 명시적인 한국 시간(KST, UTC+9) 오프셋 계산식으로 리팩토링하여 날짜 동기화 오류 해결

### 🧹 저장소 구조 최적화 (Repository Cleanup)
- **불필요한 중복 코드 베이스 제거**
  - 충돌 및 버전 혼동을 유발하던 `bible_pilsa_data_repo` 중복 폴더를 루트에서 완전 삭제하여 단일 원천(`frontend`, `backend`) 구조 확립
- **Git 추적 데이터 다이어트 및 이그노어 최적화**
  - 대용량 빌드 산출물(`.ait`) 및 비공개 로컬 캐시(`.granite/`)가 원격 저장소에 올라가지 않도록 `.gitignore`에 규칙 추가 및 기존 캐시 추적 해제
  - 불필요한 Mac OS 시스템 파일(`.DS_Store`) 일괄 완전 삭제
