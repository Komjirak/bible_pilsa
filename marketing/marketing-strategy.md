## 1. 타겟 마케팅 전략

### 1-1. 1차 타겟 프로파일

| 항목 | 내용 |
|------|------|
| 연령대 | 40~60대 기독교인 (스마트폰 이용자) |
| 직업 | 주부, 중장년 직장인, 자영업자 — 매일 큐티 습관 있거나 원하는 사람 |
| 토스 이용 패턴 | 금융 관리 주 목적, 소액 포인트 적립 미니앱 사용에 친숙한 중·고빈도 이용자 |
| 핵심 니즈 | 바쁜 일상에서도 성경 말씀을 가까이하고 싶음. 필사를 통해 믿음을 다지고 싶음. 별도 앱 설치 없이 토스 안에서 간단히 해결 원함 |

**1차 타겟 핵심 메시지 훅**

> "하루 한 절, 오늘의 말씀을 직접 써보세요."

**1차 타겟 채널 전략**

- 앱인토스 스토어 '생활·자기계발' 카테고리 자연 유입 (키워드 최적화)
- 교회·성경공부 소그룹 카카오톡·밴드 공유 바이럴 (완료 인증 이미지 공유 기능 활용)
- 기독교 커뮤니티(갓피플, 두란노, 유튜브 교회 채널) 콘텐츠 협업

---

### 1-2. 2차 타겟 프로파일

| 항목 | 내용 |
|------|------|
| 연령대 | 30대 직장인 기독교인 |
| 직업 | 교회 청장년부 소속, 주일 예배 참석하지만 평일 큐티 시간 부족한 사람 |
| 토스 이용 패턴 | 앱인토스 헤비 유저, 다양한 미니앱 탐색에 익숙 |
| 핵심 니즈 | 큐티 해야 한다는 죄책감, 성경 찾기 귀찮음, 바쁜 출퇴근 시간에 짧게 할 수 있는 영적 루틴 원함 |

**2차 타겟 핵심 메시지 훅**

> "1분이면 충분해요. 지하철에서 한 구절 필사."

**2차 타겟 채널 전략**

- 토스 피드 배너 (기독교/자기계발 관심사 타겟팅, 아침 출근 시간대)
- 인스타그램·유튜브 숏폼 (30초 필사 과정 ASMR·감성 영상)
- 앱인토스 스토어 '신규 앱' 섹션 자연 노출

## 2. 홍보 방안

| 채널 | 방법 | 예상 효과 | 우선순위 |
|------|------|---------|---------|
| 앱인토스 스토어 노출 | 생활·자기계발 카테고리 등록, 검색 키워드 최적화(성경·필사·묵상·큐티·말씀), 스크린샷 3종+필사 완료 화면 우선 노출 신청 | 신규 유입 기반 마련, 스토어 상위 노출 | P0 |
| 토스 앱 내 피드/배너 | 기독교·자기계발 관심사 사용자 대상 타겟 배너. 오전 6~8시(큐티 시간대) 집중 노출 | 기존 토스 이용자 자연 전환 | P1 |
| SNS (인스타그램) | 매일 성경 구절 카드뉴스 포스팅, #오늘의말씀 #성경필사 #큐티 #말씀묵상 해시태그 활용 | 종교 커뮤니티 유입 및 바이럴 확산 | P1 |
| 교회·소그룹 공유 바이럴 | 필사 완료 후 완성 구절 카드 카카오톡·밴드 공유 버튼 제공 → 교회 단톡방 자연 확산 | 재방문율 + 신규 유입 동시 견인 | P1 |
| 기독교 커뮤니티 협업 | 갓피플·두란노몰·크리스천투데이 등 기독교 미디어 앱 소개 기사 배포 | 핵심 타겟층 직접 도달 | P2 |
| 유튜브 숏폼 | 30초 필사 과정 ASMR 스타일 영상 제작 | 20~30대 청년 신자층 유입 | P2 |

## 3. 런치 플랜

### 소프트 런치 (런치 D-Day ~ D+14)

- **목표**: 초기 핵심 사용자 확보, 버그 수정 및 UX 개선
- **대상**: 교회·성경공부 소그룹, 기독교 유튜버 베타 체험단 (50~100명)
- **활동**:
  - 앱인토스 내부 검수 통과 후 스토어 등록
  - 소그룹 지인 공유를 통한 초기 완료 데이터 수집
  - 인앱 피드백 채널 운영 (카카오 채널 또는 이메일)

### 정식 런치 (D+15 ~ D+30)

- **목표**: 앱인토스 스토어 정식 노출 시작, MAU 1,000명 달성
- **활동**:
  - 앱인토스 스토어 '신규 앱' 섹션 노출 요청
  - 인스타그램 카드뉴스 첫 게시 (3회/주 유지)
  - 기독교 미디어 보도자료 배포
  - 토스 피드 배너 집행 검토

### 성장 단계 (D+31 이후)

- **목표**: MAU 10,000명, D7 재방문율 35% 이상
- **활동**:
  - 필사 완료 구절 공유 카드 기능 고도화
  - 주간 완주 스트릭 배지 추가
  - 시즌 마케팅 집중 (사순절·부활절·추수감사절·성탄절)
  - 교회 단위 그룹 필사 챌린지 기능 검토 (v2)
