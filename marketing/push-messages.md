# Push 메시지 — 말씀필사 (Bible-Pilsa)

> 버전: v1.0
> 작성일: 2026-03-06
> 작성 주체: MARKETING AGENT (STAGE 3-M)

---

## 작성 원칙

- 제목(title): 최대 20자
- 본문(body): 최대 50자
- 이모지 1~2개 적극 활용
- 행동 유도 문구(CTA) 포함
- 개인화 토큰 `{userName}` 활용 가능
- 앱인토스 정책 위반 표현(베팅·금융 조언·허위 과장 광고 등) 절대 사용 금지

---

## 시나리오 1: 첫 진입 유도 (신규 사용자)

```
A안)
title: ✍️ 오늘 말씀을 새겨볼까요?
body: 하루 한 절, 토스에서 바로 시작하는 성경 필사예요.

B안)
title: 📖 성경 필사, 지금 시작해요
body: 별도 앱 없이 토스에서 바로! 오늘 첫 구절을 써보세요.

C안)
title: ✨ {userName}님께 오늘의 말씀
body: 오늘의 구절이 기다리고 있어요. 지금 필사해보세요.
```

---

## 시나리오 2: 재방문 유도 (비활성 사용자 — 3일 미접속)

```
A안)
title: 📖 말씀이 기다리고 있어요
body: 3일째 쉬고 계시네요. 오늘 한 구절만 써볼까요?

B안)
title: ✍️ 잠깐, 오늘 필사는요?
body: {userName}님의 필사가 멈춰 있어요. 오늘 다시 시작해볼게요.

C안)
title: 🌅 오늘의 말씀이 도착했어요
body: 잠깐이면 돼요. 오늘 구절 하나만 새겨봐요.
```

## 시나리오 3: 특정 이벤트/시즌

### 시즌 A — 사순절 챌린지 (3~4월)

```
A안)
title: ✝️ 사순절 40일 챌린지!
body: 오늘부터 40일, 매일 한 구절로 주님과 함께해요.

B안)
title: 📖 사순절 특별 말씀 필사
body: 사순절 구절 세트가 준비됐어요. 오늘 시작하세요!

C안)
title: ✨ 오늘의 사순절 말씀
body: 매일 한 구절, 40일 완주를 향해 함께 가요.
```

### 시즌 B — 성탄절 (12월)

```
A안)
title: 🎄 성탄 말씀을 새겨봐요
body: 예수님 탄생의 기쁨을 오늘 구절로 새겨보세요.

B안)
title: ✍️ 성탄 필사 카드 만들기
body: 완성한 구절을 카드로 만들어 소중한 분께 보내봐요.

C안)
title: 🌟 오늘의 성탄 말씀
body: 특별한 성탄절, 말씀 한 구절로 마음을 채워봐요.
```
